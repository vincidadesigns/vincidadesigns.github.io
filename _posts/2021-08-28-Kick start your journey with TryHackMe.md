WHAT IS TRYHACKME ?
===================

[TryHackMe](https://tryhackme.com/) is an awesome online Cybersecurity training platform. TryHackMe makes learning comfortable by designing prebuilt courses which include virtual machines (VM) hosted in the cloud ready to be deployed. This avoids the hassle of downloading and configuring VM’s.

TryHackMe just like HTB offers multiple boxes ready to be deployed on the cloud and exploit them. Also Stand alone challenges that can be downloaded.

**MODULE 2 :: Network Fundamentals**

<img alt="" class="fo el eh eq x" src="https://miro.medium.com/max/1400/1\*G66fn8HnHZ1dzrRxikY0vw.png" width="700" height="455" srcSet="https://miro.medium.com/max/552/1\*G66fn8HnHZ1dzrRxikY0vw.png 276w, https://miro.medium.com/max/1104/1\*G66fn8HnHZ1dzrRxikY0vw.png 552w, https://miro.medium.com/max/1280/1\*G66fn8HnHZ1dzrRxikY0vw.png 640w, https://miro.medium.com/max/1400/1\*G66fn8HnHZ1dzrRxikY0vw.png 700w" sizes="700px" role="presentation"/>

This module consist of 5 rooms out of which 3 of them are subscribed only rooms, which means you need to have a tryhackme subscription.

This module mainly focuses on the core concepts of communication between the computers, network and network weaknesses. It contains hands on practice labs that could be accessed from browser.

⟴ **_Free rooms_**

⇴ What is Networking

⇴ Intro to LAN

⟴ **_Subscribed rooms_**

⇴ OSI Model

⇴ Packets & Frames

⇴ Extending Your Network

**MODULE 3 :: How The Web Works**

<img alt="" class="fo el eh eq x" src="https://miro.medium.com/max/1400/1\*0zI5IYIbFhL2hgZ8yF3LAg.png" width="700" height="380" srcSet="https://miro.medium.com/max/552/1\*0zI5IYIbFhL2hgZ8yF3LAg.png 276w, https://miro.medium.com/max/1104/1\*0zI5IYIbFhL2hgZ8yF3LAg.png 552w, https://miro.medium.com/max/1280/1\*0zI5IYIbFhL2hgZ8yF3LAg.png 640w, https://miro.medium.com/max/1400/1\*0zI5IYIbFhL2hgZ8yF3LAg.png 700w" sizes="700px" role="presentation"/>

The module consist of 4 rooms in which 2 are free and 2 are subscribed rooms.

This module focuses on the world wide web, how it functions. How DNS work and how it help you to access the internet and how you request content from a web server using HTTP protocol. It also has hands on practice labs that could be accessed in your browser.

⟴ **_Free rooms_**

⇴ DNS in Detail

⇴ HTTP in Detail

⟴ **_Subscribed rooms_**

⇴ How websites work

⇴ Putting it all together

**MODULE 4 :: Linux Fundamentals**

<img alt="" class="fo el eh eq x" src="https://miro.medium.com/max/1400/1\*1npfbJ1pyWO90TjZ2ETENw.png" width="700" height="311" srcSet="https://miro.medium.com/max/552/1\*1npfbJ1pyWO90TjZ2ETENw.png 276w, https://miro.medium.com/max/1104/1\*1npfbJ1pyWO90TjZ2ETENw.png 552w, https://miro.medium.com/max/1280/1\*1npfbJ1pyWO90TjZ2ETENw.png 640w, https://miro.medium.com/max/1400/1\*1npfbJ1pyWO90TjZ2ETENw.png 700w" sizes="700px" role="presentation"/>

The module consist 3 rooms which are totally free.

This module focuses on how to use linux operating system. All the three rooms available in this module are hands on practice labs. Which means that you can practice along with studying how to use linux operating system. In this room either you can access the labs in your browser or via SSH (secure shell protocol).

**MODULE 5 :: Windows Fundamentals**

<img alt="" class="fo el eh eq x" src="https://miro.medium.com/max/1400/1\*h24xnJeBsJeNgcLfZVCcYg.png" width="700" height="247" srcSet="https://miro.medium.com/max/552/1\*h24xnJeBsJeNgcLfZVCcYg.png 276w, https://miro.medium.com/max/1104/1\*h24xnJeBsJeNgcLfZVCcYg.png 552w, https://miro.medium.com/max/1280/1\*h24xnJeBsJeNgcLfZVCcYg.png 640w, https://miro.medium.com/max/1400/1\*h24xnJeBsJeNgcLfZVCcYg.png 700w" sizes="700px" role="presentation"/>

This module consist of 2 free rooms which can be accessed through either browser or through RDP (remote desktop protocol).

Module consist hands practice labs focusing on Windows and it’s security protocols. The module will also help you in identifying, exploiting and also to defend windows systems.
